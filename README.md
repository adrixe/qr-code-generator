# QR Code Generator (Python)
A minimal Python script that takes user input and generates a QR Code using the `qrcode` library.
> Built using just a few lines of Python code. Automatically generates unique filenames for each QR Code.

## Features
- Simple and Lightweight
- Converts any text/URL into a QR Code
- Saves QR Codes with unique filenames (e.g. `qrcode_1234.png`)
- Beginner-friendly Python Project

## Requirements
- Python 3
- `qrcode` library

## Installation
Install the required library using pip:
```bash
pip install qrcode[pil]
```
